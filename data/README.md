# Data Directory

This directory is reserved for local runtime outputs.

Generated crawl/probe/backfill artifacts are intentionally ignored by Git:

- `data/probes/`
- `data/experiments/`
- `data/lake/`
- `data/raw_snapshots/`

Keep durable findings in `docs/reports/` and source definitions in `configs/`.
Small committed examples may be placed under `data/samples/`.
